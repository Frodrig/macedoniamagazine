Capítulo 1. 
Introducción al curso de HTML

El objetivo de este curso es aprender un poquito más del mundillo HTML y dar un aspecto "pofesional" a vuestras páginas web.

Doy por supuesto que la gente que lee esto tiene unas nociones mínimas de HTML, sino es tú caso, te recomiendo que primero leas el Cursillo de HTML que hay publicado en esta revista y luego continúes con este.

Ya veréis que todo el tema de programación al lado del cliente (crear páginas HTML, Dynamic HTML, Hojas de estilo...) es bastante sencillo. Luego está el tema de programación al lado del servidor, es decir, CGI (Common Gateway Interfaces) y ASP (Active Server Pages) que requieren unas nociones de programación. En el caso de los CGI los programaremos en C y las páginas ASP las programaremos en Visual Basic .

El temario de este curso será el siguiente (posiblemente añada algún capitulo más ):

  • Mapas interactivos
  • Animaciones y transparencias
  • Creación de tablas
  • Creación de marcos (frames)
  • Incorporar sonido a una página web
  • Creación de formularios
  • Publicación y promoción de una página
  • Contadores de visitas y libros de visitas
  • Applets de java
  • Hojas de estilo
  • Introducción a Visual Basic Script
  • Introducción a JavaScript
  • Dynamic HTML
  • CGI’s
  • Páginas ASP

¿Qué se necesita para seguir este curso?

Para crear las páginas HTML necesitas un mísero editor de texto (Wordpad por ejemplo) aunque si tienes algún programa de edición de páginas HTML (Microchof FrontPage, Hot Metal, etc..) pues mejor para ti.

Para crear los CGI’s necesitas un compilador de C y para la creación de páginas ASP utilizaremos Visual InterDev (si no lo tienes no pasa nada, con el WordPad también se pueden hacer ASP’s, simplemente es por comodidad a la hora de programar).

Intentaré hacer este curso lo más ameno posible, es decir, teoría la justa y muchos ejemplos. Si tenéis alguna duda , problema, etc podéis mandarme un mail y con mucho gusto os ayudaré (si puedo , claro).
