1.- INTRODUCCION.

En esta unidad vamos a tratar de desarrollar el tema de macros. En AMI PRO le da al usuario unas macros ya preparadas, y que corresponde a lo que el usuario hace habitualmente.

2.- CONCEPTOS BASICOS.

Una macro es un programa formado por mandatos que automatizan una tarea o un grupo de tareas. Las macros de Ami Pro se crean grabando las acciones necesarias para realizar una operación o introduciendo los mandatos en un documento de Ami Pro. Al introducir las instrucciones para escribir una macro debe usar el Lenguaje de macros de Ami Pro.

Ami Pro puede ejecutar las macros automáticamente cuando:

· Pulsa el botón del ratón sobre un icono personalizado.

· Carga una aplicación o sale de ella.

· Crea un documento y selecciona una hoja de estilos.

· Selecciona un marco.

· Abre o cierra un documento.

Lotus ha desarrollado más de 40 macros para mejorar Ami Pro. Estas macros son "mini" programas opcionales que proporcionan funciones adicionales o métodos más sencillos o más rápidos para llevar a cabo determinadas tareas. Al instalar Ami Pro, estas macros se copian automáticamente en el directorio AMIPRO\MACROS.

3.- GRABACIÓN Y EJECUCIÓN DE MACROS RÁPIDA.

Es posible utilizar Grabación rápida y Ejecución rápida para grabar y ejecutar macros con rapidez. Esta función le permite automatizar su trabajo sin necesidad de grabar pulsaciones de teclas en un determinado archivo de macro.

Ami Pro guarda la macro rápida en un documento llamado SINTIT.SMM. Si utiliza Grabación rápida para grabar varias macros y selecciona Ejecución rápida, sólo podrá ejecutar la última que haya grabado.

Grabación de una macro rápida

Las macros rápidas resultan apropiadas para realizar tareas de corta duración o temporales.

1. Elija Herramientas/Macros.

2. Elija Grabación rápida.

Ami Pro muestra el mensaje "Se está grabando la macro" en la barra de estado.

3. Pulse las teclas que desee utilizar.

4. Cuando haya introducido las pulsaciones de la macro, elija Herramientas/Macro y Terminar grabación rápida.

Ratón: Pulse el ratón sobre el mensaje "Se está grabando la macro" en la barra de estado.

 

Ejecución de una macro rápida

1. Elija Herramientas/Macros.

Ratón: Pulse el ratón sobre el icono Ejecución rápida de macro.

2. Elija Ejecución rápida.

Ami Pro ejecuta la última macro rápida que haya grabado. Es posible asignar teclas rápidas a una macro rápida.

Las teclas rápidas que se pueden asignar a las macros rápidas se dividen en dos grupos: las que indican a Ami Pro que comience o termine de grabar una macro rápida y las que las ejecutan. Ami Pro usa las teclas rápidas para todas las macros rápidas que se graben y ejecuten.

4.- ASIGNACIÓN DE TECLAS RÁPIDAS A UNA MACRO RÁPIDA.

1. Elija Herramientas/Macros.

Ratón: Pulse el ratón sobre el icono Iniciar/interrumpir grabación de macro y vaya al paso 3.

2. Elija Grabar.

3. Elija Opciones.

4. Pulse las teclas que desea usar para comenzar y terminar la grabación de cualquier macro.

Puede usar CTRL, MAYUS o ambas con casi todas las teclas. No puede emplear F10, MAYUS+F1 ni MAYUS con una tecla alfanumérica. Sí podría utilizar, por ejemplo, MAYUS+F2.

5. Elija Aceptar para volver al recuadro de diálogo Grabar macro.

6. Elija Cerrar para volver al documento.

5.- EJECUCIÓN DE UNA MACRO GUARDADA EN UN ARCHIVO.

Es posible ejecutar macros que haya guardado previamente en un archivo. Si ha asignado teclas rápidas a la macro, puede utilizarlas. En caso contrario, puede utilizar la función Ejecutar.

Si ejecuta una macro creada con la Versión 1.2 de Ami Pro, que tenía funciones de macro no disponibles en la Versión 3.0, la macro seguirá funcionando. Sin embargo, si muestra y edita un archivo de macro e intenta guardarlo en la Versión 3.0, Ami Pro mostrará un mensaje de error.

1. Elija Herramientas/Macros.

Ratón: Pulse sobre el icono Ejecutar macro y especifique las opciones en el recuadro de diálogo.

2. Elija Ejecutar.

3. Especifique el nombre del archivo de macro.

Podrá seleccionar una unidad y un directorio distintos en el recuadro de lista Macros o bien introducir la vía completa de acceso al archivo de macro en el recuadro de texto Archivo de macro.

Ratón: Pulse dos veces sobre el nombre del archivo de macro para ejecutar la macro.

4. Elija Aceptar.

Ami Pro ejecuta la macro que se haya especificado.
