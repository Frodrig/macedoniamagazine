1.- INTRODUCCION.

En esta Unidad vamos a tratar de las operaciones mas habituales en AMI PRO, como pueden será: poner negrita, subrayar, cambiar tipo de letra del documento, operaciones con bloques de texto.

2.- SELECCIÓN DE UN TEXTO.

Se indica a Ami Pro que haga algo con un determinado bloque, párrafo o palabra seleccionando el texto y realizando una acción. Se puede usar tanto el teclado como el ratón para seleccionar texto.

Cuando el texto está seleccionado, puede sobreescribirlo, borrarlo, moverlo, copiarlo y modificar su aspecto cambiando su tipo de letra, alineación, sangrado, espaciado o atributos.

2.1.- Selección de texto mediante el ratón.

1. Mueva el ratón hasta que el puntero vertical se encuentre al principio del texto que desea seleccionar.

2. Arrastre el ratón hasta llegar al final del texto que desea seleccionar y suelte el botón.

Ami Pro borrará el texto seleccionado si se introduce texto nuevo o se pulsan las teclas de RETROCESO, SUPR o INTRO mientras el texto esté resaltado. Si realiza una de estas acciones sin querer, es posible que consiga recuperar el texto mediante la función Deshacer.

2.2.- Métodos rápidos para seleccionar texto mediante el ratón.

Ami Pro proporciona métodos rápidos para la selección de texto mediante el ratón.

Para seleccionar: Acción:

una palabra Pulse el ratón dos veces sobre la palabra

varias palabras Pulse el ratón dos veces y arrástrelo

una frase Pulse CTRL + el ratón una vez

varias frases Mantenga pulsado CTRL + pulse el ratón y arrástrelo

un párrafo Pulse CTRL + el ratón dos veces

varios párrafos Mantenga pulsado CTRL + pulse el ratón dos veces y arrástrelo

2.3.- Selección de texto mediante el teclado.

1. Sitúe el punto de inserción en la posición apropiada al principio o final del texto que desee seleccionar.

2. Mantenga pulsada la tecla MAYUS y pulse las teclas de movimiento del cursor.

También es posible usar una combinación de las teclas rápidas descritas en la página anterior y MAYUS para seleccionar texto. Por ejemplo, MAYUS + FIN selecciona todo el texto desde la posición del punto de inserción hasta el final de la línea.

Ami Pro borrará el texto seleccionado si se introduce nuevo texto o se pulsan las teclas RETROCESO, SUPR o INTRO mientras se encuentra resaltado. Si realiza una de estas acciones sin querer, puede que recupere el texto con la función Deshacer.

El usuario tendrá la última palabra a la hora de utilizar alguno de estos métodos.

3.- COPIAR O REPETIR UN TEXTO.

Es posible copiar texto mediante los SmartIcons, los menús o el teclado.

El texto copiado permanece en el Portapapeles hasta que se copie más texto, información distinta o una imagen.

Para copiar o repetir un texto en otro punto, se realizará los siguientes pasos :

1. Seleccione el texto que desee copiar.

2. Elija Edición/Copiar o el Smarticons Copiar.

Teclado: Pulse CTRL+INS o CTRL+C.

Ami Pro coloca una copia del texto en el Portapapeles. El texto original permanece en el documento.

    1. Sitúe el punto de inserción donde desee colocar el texto.

4. Elija Edición/Pegar o el Smarticons Pegar.

Teclado: Pulse MAYUS+INS o CTRL+V.

El contenido del Portapapeles aparece en la posición elegida.

El usuario tendrá la última palabra a la hora de utilizar alguno de estos métodos.

4.- DESPLAZAMIENTO DE TEXTO.

Es posible desplazar texto mediante los SmartIcons, los menús o el teclado.

El texto desplazado permanece en el Portapapeles hasta que se copie o corte más texto, información distinta o una imagen.

Para desplazar un texto de sitio, se realizará con los siguientes pasos :

1. Seleccione el texto que desee trasladar.

2. Elija Edición/Cortar o el Smarticons Cortar.

Teclado: Pulse MAYUS+SUPR o CTRL+X.

Ami Pro coloca el texto en el Portapepeles. Cuando se corta un texto, el texto original se borra del documento.

    1. Sitúe el punto de inserción donde desee colocar el texto.

      2. 4. Elija Edición/Pegar o el Smarticons Pegar.

      Teclado: Pulse MAYUS+INS o CTRL+V.

      El contenido del Portapapeles aparece en la posición elegida.

      4.1.- Métodos rápidos para desplazar texto

      1. Sitúe el punto de inserción en el texto que desea desplazar

      2. Si desea cambiar la posición del párrafo donde se encuentra el punto de inserción a la del párrafo anterior, pulse ALT+­.

      3. Si desea cambiar la posición del párrafo donde se encuentra el punto de inserción a la del párrafo siguiente, pulse ALT+¯.

      Ami Pro desplaza el texto a la posición deseada. Como Ami Pro no sitúa el texto desplazado o copiado mediante las teclas rápidas en el Portapapeles, no se puede pegar dicho texto.

      El usuario tendrá la última palabra a la hora de utilizar alguno de estos métodos.

      5.- CAMBIOS DE TIPO DE LETRA.

      Es posible modificar la familia de tipos de letra, su tamaño y el color aplicados a un texto seleccionado.

      Los tipos de letra de que dispone Ami Pro dependen de los tipos de letra de la impresora que se utilice. Si el nombre del botón familia situado en la barra de estado aparece en rojo (en color gris si se trata de un monitor monocromo) cuando el punto de inserción se encuentra en determinados párrafos y el botón tamaño del tipo de letra no reacciona al pulsar el ratón sobre él, el estilo de párrafo asignado a dicho texto corresponde a un tipo de letra que no se encuentra disponible en la impresora que se utiliza.

      A continuación, vamos a ver como se desarrolla la aplicación de un tipo de letra a una parte del documento:

      1. Seleccione el texto que desea modificar.

      2. Elija Formato/Tipo de letra.

      Ratón: Pulse el ratón sobre el botón familia situado en la barra de estado y elija una familia de tipos de letra. Pulse el ratón sobre el botón tamaño del tipo de letra y elija el tamaño que desea.

      3. Especifique la familia, el tamaño y el color del tipo de letra que desee aplicar al texto seleccionado.

      Es posible introducir un tamaño de tipo de letra especial en el recuadro de texto situado junto al recuadro de lista Tamaño, cuando la opción no está atenuada

      4. Elija Aceptar

      Ami Pro muestra el texto seleccionado con la familia, el tamaño y el color especificado.

    2. Si desea que el texto seleccionado vuelva a tener los parámetros de tipo de letra del estilo de párrafo, elija la opción Restaurar estilo.

6.- APLICACIÓN DE ATRIBUTOS A TEXTO.

Si lo desea, puede aplicar al texto seleccionado atributos como negrita, cursiva, subrayado o mayúsculas. Los atributos disponibles dependen de los tipos de letra de que disponga la impresora.

Para aplicar un atributo, hay que realizar las siguientes operaciones :

1. Seleccione el texto que va a modificar.

Teclado: Mantenga pulsada la tecla CTRL y pulse la letra correspondiente que desea aplicar al texto: O,N,S, U, o K para aplicar ese atributo.

    1. Elija el menú Formato.

3. Seleccione el atributo que desee aplicar.

4. Para eliminar un solo atributo del texto, seleccione el texto, elija el menú Formato y seleccione el atributo que aparezca junto a una marca de verificación. Para eliminar todos los realces y atributos del texto, elija Formato/Normal.

A continuación vamos a explicar los diferentes atributos que se le puede dar al texto:

* Normal : Elimina todos los realces y atributos aplicados al texto mediante el menú Formato. El texto seleccionado vuelve a tener los parámetros del estilo de párrafo.

* Negrita : Destaca el texto seleccionado.

* Cursiva : El texto seleccionado aparece en cursiva.

* Subrayado : Subraya el texto seleccionado, incluidos los espacios entre las palabras.

* Subrayado palabras : Subraya las palabras que componen el texto seleccionado y deja sin subrayar los espacios entre ellas.

* Letras : Asigna a las letras del texto seleccionado el tipo (mayúsculas, minúsculas, iniciales o versalitas) especificado. Cuando se anula este realce, el texto vuelve a tener las letras del estilo de párrafo.

* Mayúsculas : Convierte en mayúsculas todas las letras seleccionadas.

* Minúsculas : Convierte en minúsculas todas las letras seleccionadas .

* Iniciales : Convierte en mayúscula la primera letra de las palabras seleccionadas.

* Versalitas : Convierte en versalitas todas las letras seleccionadas. Ami  Pro transforma las mayúsculas originales del texto (primer carácter de cada una de las palabras seleccionadas) en el tamaño de letra mayúscula estándar correspondiente al tipo de letra especificado y convierte todas las letras minúsculas en un tipo de letra correspondiente a 7/10 el tamaño del tipo de letra original.

Si su impresora no puede imprimir versalitas con el tamaño exacto, Ami Pro utilizará un tamaño lo más parecido posible.

Ami Pro muestra el texto seleccionado con el atributo adecuado.

7.-PRACTICA.

 

A continuación vamos a desarrolla un práctica referente a lo que hemos explicado en el tema. Primeramente hay que escribir lo siguiente :

 

1.- Lotus Ami Pro y la solución completa de aplicaciones Lotus para Windows

 

1.1.- Interfaz común de usuario

Todos los productos de la solución completa de aplicaciones Lotus para Windows comparten similares patrones de presentación y funcionamiento, lo cual facilita enormemente su aprendizaje y utilización, tanto individual como en conjunto. Tal es el caso, por ejemplo, de los SmartIcons compartidos por todas las aplicaciones: en cuanto aprenda a utilizarlos en Lotus Ami Pro, podrá servirse de ellos en cualquiera de las demás aplicaciones Lotus para Windows.

1.2.- Herramientas para trabajo en grupo

Lotus Ami Pro ofrece una completa colección de herramientas especialmente diseñadas para la resolución efectiva de tareas en el entorno Windows, tanto a título individual como colectivo. Entre ellas, cabe citar el Gestor de aplicaciones Lotus, la función de selección múltiple y copia o las herramientas de Lotus Freelance Graphics para la elaboración de presentaciones.

 

Tareas a realizar :

1.- Poner en negrita y subrayado el punto 1.

2.- Poner en subrayado los puntos 1.1 y 1.2

3.- Poner en cursiva la palabra Smarticons.

4.- Poner en negrita las palabras que se refieren a programas informaticos.

5.- Practicar las operaciones de Copiar, Cortar y Pegar.

6.- Cambiar los tipos de letras a los titulos.
