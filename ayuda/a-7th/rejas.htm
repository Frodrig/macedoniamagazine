PUZZLE DEL ENREJADO

Para poder entrar al laberinto tienes que resolver este puzzle, colocando las piezas de tal forma que quede abierto un hueco en la rejilla.

Una única regla: sólo se puede mover una pieza cada vez, y ha de ser una pieza que esté junto al espacio libre (vertical u horizontalmente) que vamos a ocupar.

Una solución sería, si numeramos las posiciones como indica la figura: 5, 6, 3, 2, 5, 4, 1, 2, 5, 6, 3, 2, 5, 6, 3, 2, 1, 4, 5, 6, 3 y 2.
