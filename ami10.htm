Capítulo 10.
La Fusión en Ami Pro 3.1

INDICE:

1.- Introducción.

2.- Conceptos básicos de fusión.

3.- Creación de archivos de datos variables.

4.- Modificación de campos.

5.- Edición de archivos de datos variables de Ami Pro.

6.- Creación de documentos de fusión.

7.- Fusión de datos.

1.- INTRODUCCIÓN.

Por fusión se entiende la combianación de texto de dos o más archivos diferentes para crear listas de direcciones con distintas finalidades, cartas personalizadas, etc...

Lo más conocido son los mailings ; es decir, fusionar una carta con una base de direcciones para enviar por correo a sus respectivos destinatarios. Otras aplicaciones, pueden ser reporting.

2.- CONCEPTOS BÁSICOS DE FUSIÓN.

Se puede utilizar Fusión para combinar datos, como nombres y direcciones, con información de un documento de Ami Pro, como una carta o un informe. Por ejemplo, podría utilizar Fusión para enviar una carta a varias personas, pero personalizando cada carta con el nombre y domicilio del individuo. Fusión también permite crear listas de direcciones y etiquetas postales.

Para realizar una fusión, deberá seguir los pasos que se indican a continuación:

  1. Cree o seleccione un archivo de datos variables con la información variable (los nombres y direcciones).

  2. Cree o edite un documento de fusión con el texto estándar (la carta o el informe) con el que se va a combinar la información variable.

  3. Fusione e imprima la información con el documento.

Para realizar una fusión se pueden utilizar distintos tipos de archivos de datos. Los datos pueden estar almacenados en un archivo de datos variables de Ami Pro, una tabla de Ami Pro, un archivo de base de datos o un archivo de hoja de cálculo.

Es posible utilizar archivos de datos (registros) creados en otras aplicaciones importándolos, si tienen uno de los siguientes formatos:

· ASCII con campos de longitud fija (FIXED)

· ASCII con campos delimitados por comas (COMMA)

Si desea utilizar un archivo de datos ASCII con campos delimitados no por comas (por ejemplo, por comillas), deberá importar el archivo externo a un documento de Ami Pro :

 

· DIF

· Lotus 1-2-3, versiones 1, 1a y 2.x (.WKS y .WK1)

· Lotus 1-2-3, versiones 3 y 3.1, y 1-2-3 para Windows (.WK3)

· dBase, versiones III, III+ y IV

· Excel, versión 4.0 y anteriores (.XLS)

· Paradox, versiones hasta 3.5 (.DB)

· SuperCalc, versiones 3 y 4

 

3.- CREACIÓN DE ARCHIVOS DE DATOS VARIABLES.

Un archivo de datos variables consta de registros, que se organizan en campos. Si el archivo de datos contiene nombres y direcciones, cada grupo de nombre y dirección constituye un registro. Dentro del registro, el nombre, la dirección, el código postal, la ciudad y la provincia son campos. Es posible crear el archivo de datos como un documento de Ami Pro, usar datos de una tabla de Ami Pro o utilizar datos creados en una aplicación de base de datos u hoja de cálculo.

Puede crear un archivo de datos variables de Ami Pro incluso si la pantalla muestra otro documento.

1. Elija Archivo/Fusión.

Ami Pro muestra el recuadro de diálogo Bienvenido a fusión, especificando los tres pasos del proceso de fusión.

2. Elija la opción 1 para crear un archivo de datos nuevo.

3. Elija Aceptar.

Ami Pro muestra el recuadro de diálogo Archivo de datos de fusión y una lista de los documentos del directorio actual. Puesto que todavía no ha creado un archivo de datos variables, no puede utilizar ninguno de los documentos existentes.

4. Elija Nuevo.

Ami Pro abre un documento sin título y muestra el recuadro de diálogo Crear archivo de datos.

5. Para cada campo que desee crear, teclee el texto que desee utilizar como Nombres de campo y luego pulse INTRO, o bien, elija Añadir.

Los nombres de campo pueden contener letras o números, pero no deben empezar con un número ni constar de números solamente. Ami Pro sitúa los nombres de campo en el recuadro de lista Campos del archivo de datos, siguiendo el orden en que hayan sido añadidos.

6. Si necesita alterar el nombre de los campos, seleccione el campo adecuado en el recuadro de lista Campos del archivo de datos y luego elija el botón de desplazamiento adecuado.

7. Si necesita suprimir un campo, selecciónelo en el recuadro de lista Campos del archivo de datos y luego elija Suprimir.

8. Si necesita cambiar un nombre de campo, suprima uno existente y añada uno nuevo.

9. Si desea especificar delimitadores de campo y de registro especiales, elija Opciones.

El delimitador de campo marca el final de un campo (la ciudad, por ejemplo) y el delimitador de registro marca el final de un registro (un grupo de nombre y dirección, por ejemplo). Estos caracteres especiales no pueden aparecer en ningún lugar de la información.

10. Elija los delimitadores de campo y de registro deseados.

11. Elija el botón Aceptar para regresar al recuadro de diálogo Crear archivo de datos.

12. Elija Aceptar.

Ami Pro sitúa los delimitadores de campo y de registro en la primera línea del archivo de datos variables. Sitúa los nombres de campo, separados por delimitadores de campo y con un delimitador de registro al final, en la segunda línea del archivo de datos variables.

Ami Pro muestra el recuadro de diálogo Archivo de datos, que se parece a una tarjeta indexada. Los nombres de campo especificados aparecen a la izquierda de los recuadros de texto.

13. En cada recuadro de texto, teclee la información deseada y luego pulse INTRO.

Si el archivo de datos variables contiene más de diez campos, Ami Pro muestra una flecha hacia abajo en la parte inferior de la tarjeta indexada. Puede seleccionar la flecha para mostrar los nombres de campo restantes.

14. Si comete un error al escribir la información, corríjalo situando el punto de inserción en el recuadro de texto deseado y edite la información que contenga.

15. Cuando acabe de especificar la información para todos los campos, elija Añadir.

Ami Pro sitúa la información en la tercera línea del archivo de datos variables y en las siguientes y muestra un nuevo registro en blanco. Los primeros diez caracteres de la información especificada para el primer campo del registro anterior aparecen en un separador de tabulación.

16. Repita este procedimiento para cada registro que desee crear.

Ami Pro sitúa los registros en el archivo de datos variables siguiendo el orden en que los haya creado.

Puede modificar el orden de los registros llevando a cabo una clasificación alfabética.

17. Cuando ya haya creado todos los registros, elija Cerrar.

Ami Pro le muestra un mensaje solicitando que guarde el archivo.

18. Especifique el nombre de archivo deseado y elija Aceptar.

Puede visualizar registros especiales y editarlos, o bien, crear registros nuevos que desee añadir al archivo de datos variables.

 

4.- MODIFICACIÓN DE CAMPOS.

Es posible utilizar Administración de campos para insertar o renombrar ciertos campos en un archivo de datos variables que ya existe.

Para modificar un campo :

1. Elija Archivo/Fusión.

2. Elija la opción 1 para editar el archivo de datos variables.

3. Elija Aceptar.

Si el archivo de datos variables no aparece en pantalla, Ami Pro solicita el nombre del archivo. Especifíquelo y elija Edición.

Ami Pro muestra el primer registro en el recuadro de diálogo Archivo de datos.

4. Elija Adm de campos.

Ami Pro muestra el recuadro de diálogo Administración de campos, que contiene una lista de los nombres de campo del archivo de datos variables. A continuación, puede insertar un campo o renombrarlo.

Para insertar un campo en un archivo de datos variables :

Al insertar un campo, debe especificar la posición deseada para ese campo.

1. En el recuadro de texto Nombre de campo, teclee el nombre de campo que desee insertar.

2. En el recuadro de lista Campos del archivo de datos, elija el campo que ocupa la posición donde desea insertar el nuevo campo.

3. Elija Insertar.

4. Elija Antes del texto seleccionado o Después del texto seleccionado como posición para el nuevo campo.

5. Elija Aceptar para insertar el campo y regresar al recuadro de diálogo Administración de campos.

Ami Pro muestra el nuevo nombre de campo en el recuadro de lista Campos del archivo de datos, en la posición previamente determinada.

6. Elija Aceptar para regresar al recuadro de diálogo Archivo de datos.

Para asignar un nombre nuevo para un campo del archivo de datos variables

1. En el recuadro de lista Campos del archivo de datos, elija el campo que desee renombrar.

2. En el recuadro de texto Nombres de campo, teclee el nuevo nombre para el campo.

3. Elija Renombrar.

Ami Pro renombra el campo seleccionado.

4. Elija Aceptar para regresar al recuadro de diálogo Archivo de datos.

Ami Pro muestra el registro en pantalla con el nombre de campo modificado.

5.- EDICIÓN DE ARCHIVOS DE DATOS VARIABLES DE AMI PRO.

Es posible editar un archivo de datos variables modificando la información de uno o más campos, añadiendo nuevos registros o borrando registros que ya existen.

Deberá visualizar los registros antes de poder editarlos.

La edición de un archivo de datos variables de Ami Pro se debe realizar :

1. Elija Archivo/Fusión.

2. Elija la opción 1 para editar el archivo de datos variables.

3. Elija Aceptar.

Si el archivo de datos variables no aparece actualmente en pantalla, Ami Pro solicita el nombre del archivo. Especifíquelo y elija Editar.

Ami Pro muestra el primer registro en el recuadro de diálogo Archivo de datos.

4. Visualice el registro que desee editar.

5. Pulse TAB o MAYUS+TAB para desplazar el punto de inserción hacia el campo que desee editar y especifique la información deseada.

6. Elija el mandato que desee.

* Añadir : Guarda la información actual que se muestra en pantalla como un nuevo registro y muestra un registro en blanco.

* Suprimir : Borra el registro actual del archivo de datos variables y muestra el registro siguiente.

* Actualizar : Guarda toda información nueva en el registro actual y lo sigue mostrando en pantalla.

* Registro nuevo : Elimina la información en pantalla sin guardarla y muestra un registro en blanco.

7. Después de editar los registros del archivo de datos variables, elija Cerrar.

Ami Pro actualiza el registro actual y cierra el recuadro de diálogo Archivo de datos. Si había editado el archivo, Ami Pro le indica que guarde los cambios.

6.- CREACIÓN DE DOCUMENTOS DE FUSIÓN.

Después de crear el archivo de datos variables, puede crear el documento de fusión. El documento de fusión contiene el texto para la carta o el informe. Se puede crear un nuevo documento de Ami Pro o bien utilizar uno ya existente.

Para crear un documento de fusión :

1. El documento que contiene el texto estándar (la carta o el informe) debe ocupar la ventana activa y encontrarse en Modo calidad o visualizar un documento sin título.

2. Sitúe el punto de inserción en la posición donde desee que Ami Pro inserte la información que contiene el archivo de datos variables.

3. Elija Archivo/Fusión.

Ami Pro muestra el recuadro de diálogo Bienvenido a fusión, especificando los tres pasos del proceso de fusión.

4. Seleccione la opción 2 para crear un documento de fusión.

5. Elija Aceptar.

Puesto que aún no ha asignado el archivo de datos variables al documento, Ami Pro muestra un mensaje. Elija Sí para utilizar el documento actual como documento de fusión.

Si acaba de crear o editar un archivo de datos variables, Ami Pro lo asigna automáticamente al documento de fusión y muestra el recuadro de diálogo Insertar campo de fusión. Elija el nombre de campo que desee insertar en el documento de fusión y diríjase al paso 8.

Ami Pro utiliza de forma automatica el archivo de datos variables especificado cada vez que se realiza una fusión con este documento de fusión. Si Ami Pro no puede especificar el archivo de datos variables, muestra el recuadro de diálogo Archivo de datos de fusión, de manera que se podrá asignar el archivo deseado.

6. Especifique el nombre del archivo de datos variables que desee asignar al documento de fusión y elija Aceptar.

Ami Pro muestra el recuadro de diálogo Insertar campo de fusión, que contiene una lista de los nombres de campo del archivo de datos variables que haya asignado al documento de fusión.

7. Elija el Nombre de campo que desee insertar en el documento de fusión.

Cuando se realiza la fusión, Ami Pro sustituye cada nombre de campo por la información que éste contiene.

8. Elija Insertar para situar el nombre de campo en el documento de fusión.

Ami Pro muestra el nombre de campo entre paréntesis angulares, <>, en el documento. El recuadro de diálogo Insertar campo de fusión permanece en pantalla.

Ami Pro no reconoce los nombres de campo y paréntesis tecleados en el documento de fusión. Deberá insertar los nombres de campo mediante el recuadro de diálogo Insertar campos de fusión.

9. Si fuese necesario, sitúe el punto de inserción donde desee insertar otro nombre de campo.

Teclado: Pulse ALT+F6 para desplazar el punto de inserción en el documento.

Es posible insertar un nombre de campo o más en una sola línea. Si algunos registros no contienen la información referente a un campo en particular, Ami Pro no deja espacios en esa línea durante la fusión. Si el nombre de campo es el único texto de la línea, Ami Pro define el interlineado de esa línea como cero durante la fusión. Siempre que el espaciado de párrafo anterior y posterior se haya definido como cero, no aparecerán espacios de sobra en el documento impreso.

10. Repita el procedimiento hasta que haya insertado todos los nombres de campo que desee.

Teclado: Pulse ALT+F6 para regresar al recuadro de diálogo Insertar campo de fusión.

Una vez insertados los campos de fusión en el documento de fusión, se pueden cortar, copiar o mover en otras posiciones del documento. Seleccione todo el nombre de campo incluyendo los paréntesis angulares y pulse SUPR, o seleccione cortar, copiar y pegar, o Arrastrar y soltar para copiar o mover el nombre de campo a la posición deseada. Además, se pueden asignar atributos de texto y mejoras de texto a los nombres de campo utilizando estilos de párrafo, SmartIcons, la barra de estado o el menú Formato. Cuando Ami Pro inserta los datos en el documento de fusión, el texto se presenta en pantalla y se imprime con los atributos que se hayan aplicado a los nombres de campo.

11. Elija el botón de acción que desee.

* Continuar fusión : Muestra el recuadro de diálogo Bienvenido a fusión, con la opción 3 seleccionada. Ahora será posible fusionar la información.

* Cerrar : Cierra el recuadro de diálogo y vuelve al documento de fusión, para que pueda seguir editándolo.

Es posible guardar el documento de fusión y los nombres de los campos como una nueva hoja de estilos. Así se puede usar esa hoja de estilos nueva para crear otros documentos de fusión. Ami Pro utiliza de forma automática el archivo de datos variables que se ha asignado al documento de fusión.

7.- FUSIÓN DE DATOS.

Después de crear el archivo de datos variables y el documento de fusión, de especificar el archivo de datos apropiado y de insertar los nombres de campo en el documento de fusión, se puede efectuar la fusión. Podrá visualizar en pantalla, imprimir o guardar documentos fusionados.

El proceso de fusión :

1. El documento de fusión debe ocupar la ventana activa y encontrarse en Modo calidad.

2. Elija Archivo/Fusión.

3. Elija la opción 3 para fusionar e imprimir el documento.

4. Elija Aceptar.

5. Elija el tipo de fusión que desee efectuar.

* Fusionar e imprimir : Envía el documento de fusión con la información que contienen los registros directamente a la impresora.

* Fusionar, ver e imprimir : Muestra el documento de fusión con la información de un registro de manera que se puede ver y modificar antes de enviarlo a la impresora. Las modificaciones que realice afectarán a futuros documentos fusionados.

* Fusionar y guardar como : Guarda el documento de fusión con la información de un registro en el archivo de Ami Pro que se especifique, para poderlo editar o imprimir más tarde.

En el documento de fusión, los nombres de campo deben encontrarse en el texto principal para poder utilizar Fusionar y guardar como. Dichos nombres no pueden encontrarse dentro de marcos, tablas, notas a pie de página, cabeceras ni pies de páginas. Además, el documento de fusión no debe contener imágenes.

6. Para especificar el número de copias, el grupo de páginas y otras opciones de impresión, elija Opcs de impresión.

7. Especifique las opciones apropiadas.

8. Elija Aceptar para regresar al recuadro de diálogo Fusión.

9. Elija Aceptar para iniciar la fusión.

Ami Pro crea los documentos con la información que contienen los registros y los muestra en pantalla, los imprime o los guarda en un archivo.
