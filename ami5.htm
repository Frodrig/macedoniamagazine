1.- INTRODUCCIÓN.

Con esta unidad pretendo dar explicaciones a temas no muy habituales en la utilización del AMI PRO como pueden ser poner algún efecto especial, formatear parrafos distintos como plantilla del primero.

2.- EFECTOS ESPECIALES.

Al texto seleccionado pueden aplicarse realces de texto como superíndices, subíndices, doble subrayado y caracteres de tachado y superpuestos.

Vamos a ver como se aplica un efecto especial :

  1. Seleccione el texto que desea modificar.

2. Elija Formato/Efectos especiales.

3. Elija el realce correspondiente :

* Superíndice : Sitúa el texto seleccionado en una posición ligeramente más elevada que el resto del texto, con un tipo de letra correspondiente a 7/12 el tamaño del tipo de letra original. Los superíndices se utilizan para las notas al pie.

* Subíndice : Sitúa el texto seleccionado en una posición ligeramente por debajo del resto del texto, con un tipo de letra correspondiente a 7/12 del tamaño del tipo de letra original. Utilice Ecuaciones para crear ecuaciones matemáticas que contengan superíndices y subíndices.

* Doble subrayado : Sitúa dos líneas debajo del texto seleccionado.

* Tachado : Muestra e imprime el texto seleccionado tachado con una línea recta.

* Carácter superpuesto : Muestra e imprime el carácter introducido en el recuadro de texto sobre el texto seleccionado. Puede utilizarse cualquier carácter ANSI o ASCII. El carácter superpuesto que se especifica utilizará el mismo tipo de letra que el texto seleccionado.

4. Elija Aceptar.

Ami Pro muestra el texto seleccionado con los realces de texto especiales seleccionados.

5. Para eliminar los Efectos especiales, seleccione el texto, elija Formato/Efectos especiales, anule la selección del realce correspondiente y elija Aceptar.

3.- FORMATO RÁPIDO.

Es posible utilizar Formato rápido para extraer la información sobre formato de texto de un texto seleccionado y aplicarlo a otro texto, al texto de una tabla o de un marco que se encuentre en el documento principal. El formato de texto es cualquier familia de tipos de letra, tamaño de letra, color, atributo (negrita, cursiva, subrayado, subrayado palabras), mayúsculas o efectos especiales que aplica al texto seleccionado utilizando SmartIcons, la barra de estado o el menú Formato.

Así mismo, puede utilizar Formato rápido para extraer información sobre el formato de estilo de párrafo.

Debe tener un ratón para aplicar la información sobre formato extraída usando Formato rápido.

Para el uso de Formato rápido en texto, se debe hacer lo siguiente :

1. Seleccione el texto que dispone del formato de texto deseado.

2. Elija Formato/Formato rápido

Ami Pro cambia la forma del puntero del ratón a la de un pincel y puede extraer cualquier información sobre el formato de texto. Si no ha aplicado ningún tipo de letra utilizando SmartIcons, la barra de estado o el menú Formato, Ami Pro extrae la información sobre el tipo de letra del estilo de párrafo. Si selecciona todo un párrafo, Ami Pro muestra el recuadro de diálogo Formato rápido. Elija Sólo tipo de letra y atributos de texto y, a continuación, Aceptar.

3. Seleccione el texto al que desea aplicar el formato de texto y suelte el botón del ratón.

El texto aparecerá con el formato que le ha aplicado. Ami Pro añade el formato de texto a cualquier formato aplicado con anterioridad. Por ejemplo, si extrae el atributo de negrita y lo aplica a un texto en cursiva, éste aparecerá en negrita y cursiva.

4. Si desea aplicar el formato de texto extraído a otro texto, vuelva a realizar el paso 3.

5. Cuando no desee aplicar el formato de texto, elija Formato/Formato rápido.

Teclado: Pulse ESC. Ami Pro cambia el puntero del ratón a cursor vertical.
