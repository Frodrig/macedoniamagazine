 PARTE 1: DETECCIÓN DE COLISIONES

Introducción

Actualmente cualquier sistema de animación interactivo contiene una gran cantidad de entidades virtuales que se mueven y interactúan entre ellas. En estas animaciones no se puede predecir con anterioridad cu√°l será el comportamiento del usuario y dichas entidades, por lo que será imprescindible ir creando la animación en tiempo de ejecución. Teniendo en cuenta que para que el usuario acepte la animación como una simulación a tiempo real la imagen mostrada tiene que actualizarse por lo menos unas 50 veces por segundo, esto nos deja, en el mejor de los casos, con sólo 20 milisegundos para realizar la actualización de todas las entidades de la simulación y representar la escena.

 

No cabe duda de que una aplicación de este tipo tiene que disponer de un sistema de visualización muy rápido; pero no se debe olvidar que la aplicación contiene una gran cantidad de entidades que necesitan ser actualizadas constantemente y cuyo comportamiento no se puede predecir al ser una aplicación interactiva. Con el tiempo estas entidades pueden llegar a penetrar unas con las otras, dando una muy mala simulación al usuario. Por lo tanto la aplicación deberá tener un sistema de tratamiento de colisiones lo suficientemente rápido como paro no arruinar el rendimiento a tiempo real deseado.

 

Este sistema de tratamiento de colisiones se divide en dos componentes. El primero es un algoritmo de detección de colisiones, el cual es el responsable de encontrar aquellas entidades cuyas superficies hayan empezado a penetrar. Si el algoritmo de detección encuentra alguna entidad en este estado, entonces el sistema de tratamiento de colisiones llama al segundo componente, un algoritmo de reacción a las colisiones. Este algoritmo tiene la función de corregir el comportamiento de las entidades de forma que dejen de penetrar. Esta corrección puede implicar la aplicación de ciertas leyes de la dinámica, como por ejemplo hacer que una entidad rebote.

 

El objetivo de esta parte del artículo es dar un modelo del algoritmo de detección que se pueda aplicar a cualquier aplicación que tenga los requerimientos anteriores.
